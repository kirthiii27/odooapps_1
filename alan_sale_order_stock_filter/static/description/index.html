Sale Order Stock Filter for Odoo 18

Show Available Quantity & Hide Out-of-Stock Products in Sale Order

This module improves the Odoo sales process by ensuring that only products with available stock are shown when creating a Sale Order. It prevents overselling, enhances order accuracy, and displays available stock right in the product dropdown of the sale order line.

🔑 Key Features:

  • ✅ Display on-hand stock quantity in Sale Order lines
  • ✅ Automatically hides products with zero or negative quantity
  • ✅ Prevents accidental overselling
  • ✅ Improves inventory management and order accuracy
  • ✅ Seamless integration with Sales and Inventory modules
