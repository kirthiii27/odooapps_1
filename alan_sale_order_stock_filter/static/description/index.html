📦 Module Overview: Sale Order Stock Filter for Odoo

Show Available Quantity & Hide Out-of-Stock Products in Sale Order

This module enhances the sales experience by allowing only products with available stock to be selected in Sale Order lines. It improves inventory accuracy and helps businesses avoid overselling by clearly showing on-hand quantity during product selection.

🚀 Why Use This Module?

  • 🚫 Eliminate overselling with real-time stock visibility.
  • 📉 Prevent selection of out-of-stock products in Sale Orders.
  • ✅ Ensure accurate inventory usage and order fulfillment.

✅ Key Features:

  • ✅ Show available quantity in Sale Order product dropdown
  • ✅ Filter out products with zero or negative stock
  • ✅ Auto-update display name with quantity info
  • ✅ Prevent selection of unavailable products
  • ✅ Seamless integration with Sales & Inventory apps
